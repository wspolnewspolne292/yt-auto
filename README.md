# YouTube Shorts Automation

Projekt Python do w pełni automatycznego generowania i publikowania viralowych shortów na YouTube.

## Struktura projektu
- ai_script/ — generowanie pomysłu i skryptu shortsa
- ai_tts/ — generowanie głosu lektora
- ai_video/ — generowanie obrazu/wideo
- video_assembly/ — montaż wideo
- youtube_upload/ — automatyczna publikacja shortsa

## Wymagania
Wszystkie zależności znajdują się w requirements.txt

## Użycie
1. Skonfiguruj klucze API (OpenAI, ElevenLabs, YouTube Data API itp.)
2. Uruchom główny skrypt: `python main.py`
